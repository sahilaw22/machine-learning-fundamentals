## Getting Started


### Prerequisites
To run the notebooks and experiments in this repository, you'll need:
- Python 3.10+
- Jupyter Notebook or Jupyter Lab
- Basic understanding of Python programming

### Installation
```bash
# Clone the repository
git clone https://github.com/sahilaw22/machine-learning-fundamentals.git

# Navigate to the project directory
cd machine-learning-fundamentals
```

# Usage


Open the notebooks in `ML Labs Python Notebooks/` using Jupyter Notebook or Jupyter Lab to explore and run the experiments interactively.

PDFs in `Experiments/` provide supporting theory and documentation for each lab.
