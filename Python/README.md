## 📁 Programs

### Practical 1: Basic Data Types — `Practical_1.ipynb`

**What it does:**
- Demonstrates the usage and characteristics of Python's built-in data types.
- Provides hands-on examples of numeric operations, string manipulation, and collection handling.

**Concepts covered:**
- **Numeric Types:** `int`, `float`, `complex`
- **Sequences:** `str` (Strings), `list` (Lists), `tuple` (Tuples)
- **Sets:** `set` (Unordered collections with no duplicates)
- **Mappings:** `dict` (Key-value pairs)
- **Booleans:** `bool` (Truthiness and falsiness)

**Sample Snippet:**
```python
# List vs Tuple
my_list = [1, 2, 3]    # Mutable
my_tuple = (1, 2, 3)   # Immutable

# Dictionary access
d = {1: 'Geeks', 'name': 'For', 3: 'Geeks'}
print(d.get("name"))   # Output: For
```
